This repository contains a sample Employee Management Dashboard built with Angular 17+, TypeScript, and Angular Material. It features a complete employee CRUD system with Material Design tables, add/edit/delete forms using both template-driven and reactive approaches with full validation, employee detail views, a custom department filter pipe, high-salary highlight directive, HTTP request logging interceptor, and route guard simulation for protected routes. The modern standalone component architecture uses app.config.ts for bootstrapping, lazy loading via loadComponent, and EmployeeService with HttpClient/RxJS for all data operations against a mock JSON Server API running at http://localhost:3000/employees.
